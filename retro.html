THE PROJECT

A comprehensive examination of the internal web application


For a usability course project, I worked in a team of four UX-focused students to provide usability consulting service for RetroQuest -- an open source web application developed by FordLabs for running weekly retrospecitves.

With no current intention to push RetroQuest to the market, the web application’s current target users are internal teams at Ford. With only one designer in charge of the UX of the application, the development team is seeking consulting from our team to help them thoroughly evaluate the application's usability.


INTERACTION MAP

First exploration of the application


To start out, we examined the application by exploring all possible elements on the UI and possible interaction. After documenting all interactions possible and bugs, we then generated an interaction map that provides a clear overview of the structure of the application.
