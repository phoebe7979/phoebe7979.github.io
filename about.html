About this site.
The portfolio website has been a ongoing side project for me, because I actually enjoy building my own site!

The site is written in HTML, CSS, and some JavaScript, with a lot of help from W3Schools and Stack Overflow. My tools include Atom and Github.

Recently, it has gone through a revamp process starting from September, 2019, for which I have added some cool JS effects such as the scroll progress bar, as well as more content, to make it feel more like me.

Please feel free to email me if you spot any bugs or have any suggestions! And of course, this website will keep evolving.
